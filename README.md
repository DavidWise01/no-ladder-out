# NO LADDER OUT · Chutes & Ladders is a Markov chain

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **BANANA**.

NO LADDER OUT — Chutes & Ladders, the toddler board with ZERO decisions, and that is exactly what makes it a PURE absorbing Markov chain: no strategy layer muddies the map. Squares are states, the spinner is the transition kernel, each chute and ladder is one off-diagonal entry, square 100 is the single absorbing state. ⚑ GREEN (node-verified two ways): the fundamental matrix N=(I−Q)⁻¹, N·1, gives expected rolls to finish from square 0 = 39.2251 (exact-landing; 35.8349 overshoot-wins), and a 400k-trial simulation returns the same (39.20 / 35.81); 100 is the unique absorbing state, I−Q invertible (termination with probability 1). The live Monte-Carlo histogram converges to 39.2251. AMBER: the exact number depends on THIS board's chute/ladder placements + the exact-landing convention; editions/houserules shift it. RED: the wrapper is clean precisely because the game is trivial — the moment a game admits a DECISION it stops being a plain Markov chain (you'd need a Markov decision process + strategy); this models the one-player, no-choice case honestly and no further.
